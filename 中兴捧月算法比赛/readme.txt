一、运行坏境：Windows10 、Windows7

二、工程编译环境：VS2015

三、测试说明

1、测试文件需要放在TestData文件夹下，TestData文件夹必须与可执行文件在同一目录之下；
2、运行时需要输入本次测试所使用测试文件的文件名（包括扩展名)；

3、可运行已提供用例之外的测试用例，需保持用例例格式，并将用例放入TestData文件夹；

四、测试用例文件格式说明

图类型（0无向图/1有向图） 顶点个数  边数   阈值（允许经过的最大节点数目）

边1端点1   边1端点2   边1权值
...

边n端点1   边n端点2   边n权值

必经点数 必经边数 禁止经过边数

起始点   终止点

必经点1   必经点2 ...   必经点n


必经边1端点1    必经边1端点2
...

必经边n端点1 必经边n端点2


禁止边1端点1    禁止边1端点2
...

禁止边n端点1 禁止边n端点2

注：回车和换行符不做限制
